# Wild Wonders

Wild Wonders is a Progressive Web App (PWA) designed to educate users about wild animals with engaging audio, images, and descriptions.

## Features
- Select from a dropdown list of wild animals
- Canvas displays the selected animal’s image
- Audio plays fun facts or animal sounds
- Fully installable and offline-capable

## Installation
1. Open `index.html` in Chrome
2. Click the install icon or menu > Install App
3. App works offline once installed

## Folder Structure
- `index.html` — main app
- `data.json` — animal data
- `script.js` — handles data and display
- `manifest.json` — PWA manifest
- `service-worker.js` — offline support
- `assets/` — images, audio, icons, and screenshots
